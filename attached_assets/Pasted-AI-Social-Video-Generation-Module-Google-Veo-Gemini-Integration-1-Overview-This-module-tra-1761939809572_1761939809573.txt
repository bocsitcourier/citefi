AI Social Video Generation Module (Google Veo + Gemini Integration)
🚀 1️⃣ Overview

This module transforms AI-generated articles or posts into 1-minute TikTok and YouTube Shorts videos with natural voiceovers, captions, music, and visuals using Google Veo for video generation and Gemini for script and narration logic.

Core Objectives

Generate short-form videos (≤ 60 s) for TikTok, YouTube Shorts, Reels, and Pinterest.

Voiceover in male or female conversational tone.

Include captions, hashtags, emojis, and optional background music.

Allow editing, regeneration, preview, download, and auto-sharing.

Built as a stand-alone SSR module deployable in Next.js + Node.js hosted on DigitalOcean.

🧱 2️⃣ System Architecture
Frontend

Next.js SSR Dashboard

Inputs: topic, title, tone, mood, location, platform, duration, style.

Fields for user’s connected social handles.

“Generate Video” button triggers backend generation.

Video preview component with playback, edit, share, download options.

Backend

Node.js Express API

Handles Gemini and Veo API calls.

Manages file storage, metadata, and progress updates.

Streams progress events to frontend via WebSocket or SSE.

Google Veo API

Generates cinematic, platform-optimized short videos.

Accepts script + scene descriptions + voice guidance + visual cues.

Gemini API

Summarizes article → creates 60 s script.

Generates voiceover and caption text.

Suggests emojis and hashtags.

Storage: DigitalOcean Spaces (S3-compatible)

Database: PostgreSQL + Prisma ORM

🔗 3️⃣ Data Flow

1️⃣ User Input → Gemini Prompt

User enters:
Topic, Title, Mood, Tone, Location, Platform, Duration, Style


2️⃣ Gemini Script Generation

Produces a concise 60 s script, divided into scenes with voice cues, captions, and visual hints.

3️⃣ Gemini TTS

Converts the script into a natural-sounding narration (female or male).

4️⃣ Veo Video Generation

Input: script + voice + scene descriptions.

Veo outputs a cinematic .mp4 ready for the target platform (aspect ratio 9:16).

5️⃣ Store & Render

Upload to DigitalOcean Spaces.

Save DB entry with video URL, transcript, and share metadata.

6️⃣ Playback & Share

Frontend displays preview.

User can edit captions, regenerate audio, download, or auto-share to connected social accounts.

🧩 4️⃣ API Specification
POST /api/video/generate

Body

{
  "articleId": "uuid",
  "platform": "tiktok",
  "tone": "business-friendly",
  "mood": "energetic",
  "voice": "female",
  "location": "Boston, MA",
  "duration": 60
}


Process

Fetch article text.

Send to Gemini → structured 60 s script.

Generate voiceover (Gemini TTS).

Pass to Google Veo → video creation.

Upload to storage.

Return video metadata and progress ID.

Response

{
  "status": "processing",
  "jobId": "uuid",
  "estimatedTime": "45s"
}

GET /api/video/status/:jobId

Returns generation progress from Veo API.

POST /api/video/share

Uploads finished video to connected TikTok, YouTube, or Instagram via OAuth tokens.

DELETE /api/video/:id

Deletes both the DB record and the file in DigitalOcean Spaces.

🧠 12️⃣ Example Gemini → Veo Pipeline
async function generateSocialVideo(articleId, opts) {
  const article = await db.article.findUnique({ where: { id: articleId } });

  const script = await gemini.generateVideoScript(article.text, opts);
  const tts = await gemini.generateVoice(script.voiceLines, opts.voice);
  const veo = await googleVeo.generateVideo({
    script: script.text,
    voiceover: tts.url,
    tone: opts.tone,
    mood: opts.mood,
    platform: opts.platform
  });

  await storage.save(veo.videoUrl, `videos/${articleId}.mp4`);
  return veo.videoUrl;
}
