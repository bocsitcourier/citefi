The worker.ts Audit & Fix
Search for the section where the worker processes the batch data (usually around the GPT-4 response handling). You need to ensure the generationParams are extracted correctly before calling the new slug-map-injector.

Apply this fix to the generation loop:

TypeScript
// Inside your worker's main loop/function
const [batch] = await db.select().from(batches).where(eq(batches.id, job.batchId));

if (!batch) throw new Error("Batch context missing");

// 1. SAFE EXTRACTION: Cast the JSONB column to a usable object
const params = (batch.generationParams as { 
  geographicFocus?: string; 
  coreTopic?: string; 
  businessName?: string;
}) || {};

// 2. PRE-PROCESS: Build the fallback list using the extracted params
const fallbackTerms = buildFallbackTerms({
  coreTopic: params.coreTopic,
  geographicFocus: params.geographicFocus,
  businessName: params.businessName,
  geminiKeywords: (batch.metadata as any)?.keywords || []
});

// 3. GENERATE: (Existing GPT-4 logic happens here...)
const gptFinalHtml = await generateArticle(batch, facility); 

// 4. INJECT: The "Architect's Chokepoint" - Happens AFTER GPT is done
const { entries } = await buildSlugMap(batch.teamId, batch.targetUrl || "", fallbackTerms);
const finalLinkedHtml = injectLinksFromSlugMap(gptFinalHtml, entries, facility.name);

// 5. SAVE: Store the hyperlinked version in the DB
await db.update(articles)
  .set({ contentHtml: finalLinkedHtml })
  .where(eq(articles.id, articleId));

console.log(`🚀 [Worker] Generation & Hyperlinking complete for ${facility.name}`);
Why this prevents future "Link-less" articles:
Explicit Typing: By defining params immediately after the DB fetch, you ensure geographicFocus isn't lost in the "JSONB void."

Sequential Integrity: The hyperlinking happens after the GPT-4 string is returned but before the DB save. This means the version stored in your database is always the "finished product."

Fail-Safe: If for some reason the site_map crawl fails, the fallbackTerms (which now correctly include the city/town name) will step in and create those vital local links.
