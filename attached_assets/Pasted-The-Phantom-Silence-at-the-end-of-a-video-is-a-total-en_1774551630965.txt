1. The Voiceover Timeout Fix: "Master Duration" Sync
The reason your voice ends before the video is likely because your video clips (slideshow) are calculated at a fixed length (e.g., 60s), but the AI-generated voiceover fluctuates based on speaking speed.

The Fix: We must calculate the Actual Audio Duration first, then tell FFmpeg to stretch or loop the video to match it exactly.

TypeScript
// lib/video-utils.ts
import { exec } from 'child_process';
import util from 'util';
const execPromise = util.promisify(exec);

async function getAudioDuration(filePath: string): Promise<number> {
  // Use ffprobe to get the exact millisecond duration of the voiceover
  const { stdout } = await execPromise(
    `ffprobe -v error -show_entries format=duration -of default=noprint_wrappers=1:nokey=1 ${filePath}`
  );
  return parseFloat(stdout);
}

2. The "Spell-Check" Guard (The Sanitizer)
To fix misspellings, we don't just rely on the AI. We run the text through a Dictionary Guard that catches common AI hallucinations or brand name errors before they reach the FFmpeg drawtext filter.

TypeScript
const BRAND_DICTIONARY: Record<string, string> = {
  "Replit": "Replit",
  "Neondb": "Neon",
  "Postgresql": "PostgreSQL",
  // Add your specific brand terms here
};

export function sanitizeOverlayText(text: string): string {
  let sanitized = text;
  
  // 1. Fix Brand Spellings
  Object.entries(BRAND_DICTIONARY).forEach(([wrong, right]) => {
    const regex = new RegExp(wrong, 'gi');
    sanitized = sanitized.replace(regex, right);
  });

  // 2. Remove AI "Thinking" artifacts
  sanitized = sanitized.replace(/["'“”]/g, ''); // Strip quotes that break layouts
  sanitized = sanitized.trim();
  
  return sanitized;
}
3. The "Missing Logo" Overlay Logic
If you don't see the logo, it's usually a "Z-index" issue in FFmpeg—the text or video is being drawn over it. We need to place the logo as the final layer in the filter complex.

The Platinum Command with Logo:

TypeScript
const LOGO_PATH = path.join(process.cwd(), 'assets', 'logo.png');

const args = [
  '-i', videoInput, 
  '-i', audioInput,
  '-i', LOGO_PATH, // Input 2: The Logo
  '-filter_complex', [
    // [0:v] is video, [2:v] is logo. Overlay logo at top-right (w-overlay_w-20)
    `[0:v][2:v]overlay=W-w-40:40[v_with_logo]`,
    // Now take [v_with_logo] and add the text
    `[v_with_logo]drawtext=fontfile='${FONT}':textfile='${TEXT_FILE}':x=(w-text_w)/2:y=(h-text_h-100):...`
  ].join(';'),
  '-c:a', 'aac', '-shortest', // Sync to the audio
